# Project - MNIST

## Description

### Objective

Implement a simple Convolutional Neural Network (CNN) for the MNIST dataset.

The objective of this project is to:

1. Classify handwritten digits using the famous MNIST data set.
2. Segment the digits from the background

### Data

The MNIST dataset is a collection of handwritten digits, where each digit is represented by a 28x28 pixel image. The dataset is split into three parts:

- Training set: 60,000 images
- Validation set: 10,000 images
- Test set: 10,000 images
